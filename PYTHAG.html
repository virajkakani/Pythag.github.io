Features

Math Types

Built-in fraction, complex, matrix, and polynomial types with exact arithmetic via GMP.

PEMDAS Aware

Operator precedence follows standard math rules. Use ^ for exponentiation naturally.

Math Functions

sin, cos, sqrt, ln, gcd, nthroot, and many more built in.

Clean Syntax

Typed variables, functions, classes with access modifiers, and readable control flow.

Example

# Pythag example program fraction f = fraction(1, 3); complex c = complex(3, 4); float x = 2.0; print(f); # 1/3 print(magnitude(c)); # 5.0 print(sqrt(x) + sin(0)); # 1.41421 int function factorial(int n) { if (n <= 1) { return 1; }; return n * factorial(n - 1); }; print(factorial(10)); # 3628800

Variables & Constants

type variable_name; const type variable_name = value;

Built-in types: int, float, char, string, bool, fraction(n,d), complex(re,im), vector, matrix, polynomial, list, set, map, tuple

Built-in constants: pi, e, i, true, false

Functions

# Block form return_type function name(params) { # body }; # Expression form return_type function name(params) = expression;

Classes

class ClassName { # use private, public, or protected modifiers };

Control Flow

# If / elif / else if(condition1){ ... } elif(condition2){ ... } else{ ... }; # For loop for (i in range(min, max); i += step){ ... }; # While loop while(condition){ ... } else{ ... };

Boolean operators: and, or, not

Math Operators & Functions

Follows PEMDAS. Use ^ for exponentiation.

Operators: + - * / % ^  |  += -= *= /= %=  |  ++ --  |  == != > < >= <=

Functions: sqrt, pow, nthroot, gcd, sin, cos, tan, sec, csc, cot, ln, log, exp, abs, min, magnitude, arg, conj, real, imag, phase, polar, rect

Built-in Type Functions

  • String: length, substring, index_of, replace, to_upper, to_lower, trim, split, join
  • List: length, append, remove, index_of, sort, reverse, map, filter, reduce
  • Set: add, remove, contains, union, intersection, difference
  • Map: put, get, remove, contains, keys, values, items
  • Tuple: length, get, set, map, filter, reduce
  • Vector: +, -, dot, cross, magnitude, normalize, angle
  • Matrix: +, -, *, transpose, determinant, inverse, eigenvalues, eigenvectors
  • Polynomial: add, subtract, multiply, power, factor, gcd, divide, derivative, integral, evaluate
  • Function: compose, partial, curry, uncurry, derivative, integral, evaluate

General Built-ins

print, input, type, len, range, enumerate, zip, map, filter, reduce

Compiling & Running

# Compile your_file.ptg PYTHAGcompileTo your_file # Run (Mac / Linux) ./your_file # Run (Windows) .\your_file

Pre-download Checklist

  • C++ Compiler: Ensure you have g++ or clang installed.
    (Mac: xcode-select --install)
  • Math Engines: You must have GMP and MPFR installed for high-precision math.
    Mac: brew install gmp mpfr | Linux: sudo apt install libgmp-dev libmpfr-dev
  • Terminal Access: Pythag is a command-line tool. You'll need to use Terminal (Mac/Linux) or CMD/PowerShell (Windows).

Need the raw libraries? Download them from the official GMP and MPFR websites.
